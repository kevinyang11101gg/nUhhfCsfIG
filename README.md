# Python热门考研专业数据分析系统简介

## 核心功能
- 登录注册
- 考研数据可视化大屏
- 考研成绩管理
- 用户管理
- 系统管理
- 个人信息管理

## 适用场景
- 考研培训机构
- 教育科研机构
- 学生个人

## 技术特点
- 使用 Django 作为后端框架
- 采用 MySQL 数据库进行数据存储
- 前端采用 HTML、CSS 和 JavaScript 进行界面展示
